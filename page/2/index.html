Ubuntu Install GitLab

2015-12-02

按照官方指导进行安装
1、先确定是64Bit Ubuntu 14.04(自己是在这个环境测试的，官方推荐Ubuntu版本)
2、

apt-get update

3、
apt-get install curl openssh-server ca-certificates

如果没有SMTP服务器，需在本机搭建安装，install postfix
4、
curl https://packages.gitlab.com/install/repositories/gitlab/gitlab-ce/script.deb.sh | sudo bash
sudo apt-get install gitlab-ce

文件有点儿大，如果网络不好，耐心等待
5、安装完成之后
gitlab-ctl reconfigure

6、官方的说法是安装完成之后可以直接访问，但是如果之前已经有站点部署，需要修改配置文件，一般在/opt/var/gitlab/nginx，修改gitlab-http.conf
7、执行gitlab-ctl reconfigure
默认用户名:root
默认密码:5iveL!fe
