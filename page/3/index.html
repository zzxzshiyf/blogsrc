Install OSSEC Agent Record

2015-11-14

在CentOS6.5上安装OSSEC记录
需要的组件:MySQL、Apache\Nginx、Postfix
首先安装服务端，官网下载OSSEC Tar包
解压，进入src目录，执行 make setdb 编译需要的MySQL环境
执行install.sh，根据提示一步步进行
编译完成之后，执行bin/management_agent 添加Agent
拷贝秘钥
安装客户端，直接执行install.sh
编译完成后，执行bin/management_agent 导入秘钥
OSSEC服务端会开启UDP 端口1514和514
注意IPTABLES规则

Install Wordpress On My Vps

2015-11-11

记录一下，也是折腾了小半天
其实很简单，如果你不用源码安装Nginx或Apache或MySQL
1、下载最新版的Wordpress
2、创建数据库，修改Wordpress目录中得wp-config-sample.php文件，记得先 cp 成wp-config.php
3、安装PHP环境，安装Nginx，安装PHP5-gd、php5-curl
4、修改Nginx站点配置文件，指向Wordpress目录，一般在/var/www/wordpress/
5、访问http://your.domain.name/wp-admin/install.php
6、That’s OK.
